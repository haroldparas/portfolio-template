## Portfolio Starter

A modern portfolio foundation built with React 19, Next.js, TypeScript, and Tailwind CSS.

It includes:

- An editorial-style homepage with hero, work, process, and contact sections.
- Responsive layouts and reusable content blocks you can customize quickly.
- A neutral visual system with warm surfaces, clear contrast, and subtle motion.

## Getting Started

Install dependencies and start the development server:

```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) in your browser.

## Scripts

- `npm run dev` - start the development server.
- `npm run build` - create a production build.
- `npm run start` - run the production build.
- `npm run lint` - run ESLint.

## Customize

- Replace the placeholder name, projects, and contact links in `src/app/page.tsx`.
- Update metadata in `src/app/layout.tsx`.
- Tune the color palette and typography in `src/app/globals.css`.

## Deployment

Deploy with your preferred Next.js host after running a production build.
